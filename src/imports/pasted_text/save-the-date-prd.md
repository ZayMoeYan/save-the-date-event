# Product Requirements Document (PRD)

## Save The Date Event Planning and Decoration Website

---

# 1. Project Overview

**Business Name:** Save The Date Event Planning and Decoration

**Website Type:** Informational / Portfolio

**Primary Goal:**
The website will present Save The Date as a professional event planning and decoration brand. It will showcase services, portfolio projects, testimonials, and provide easy ways for potential clients to contact the team.

**Main Objectives**

* Increase brand awareness
* Showcase past events and decoration projects
* Provide business information
* Allow potential clients to submit inquiries
* Support Myanmar and English languages

---

# 2. Target Audience

**Primary Audience**

* Couples planning weddings
* Recently engaged individuals
* Age range: **25–45**
* Looking for:

  * Wedding planning
  * Event decoration
  * Event coordination
  * Inspiration for events

**Secondary Audience**

* Corporate companies planning events
* Birthday celebration organizers
* Families planning special celebrations

# 6. Multilingual Support

Languages Supported:

* Myanmar
* English

Implementation:

* Language switch toggle
* Content translation for both languages

Be Inculude **modern animations and smooth transitions** across the entire website to improve user experience.


### Page Load Animation

• When users first enter the website, apply a **smooth fade-in effect** to the whole page
• Keep it subtle and fast (around 0.3s – 0.5s)

---

### Scroll Reveal Animation

• When scrolling down, content should **gradually appear with fade + slide-up effect**

Apply to:

• Sections
• Services
• Images
• Text blocks

Animation behavior:

• Start slightly below (translateY)
• Fade in while moving up
• Trigger when element enters viewport

---

### FAQ Dropdown Animation

• FAQ accordion should have **smooth open/close animation**

Behavior:

• Expand with **height transition**
• Content fades in when opening
• Smooth collapse when closing

---

### Hover Effects

• Buttons → slight scale + red color transition
• Cards → soft lift (translateY) + shadow

---

###  Page Transition

• Add **smooth transition between pages** (fade or slight slide)
• Avoid sudden hard cuts

---

###  Animation Guidelines

• Keep animations **simple and clean**
• Duration: **0.3s – 0.6s**
• Avoid heavy or distracting effects
• Maintain a **premium and smooth feel**

---

### Goal

Make the website feel:

• Modern
• Smooth
• Interactive
• High-quality user experience
